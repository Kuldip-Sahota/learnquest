About LearnQuest

LearnQuest is a web application built to manage and explore courses. Students can view available courses and track their learning, while instructors can add new courses and organize content.

Features:

  • Browse courses with filtering and sorting
  • Add new courses with name, category, and difficulty level
  • Responsive design using Bootstrap 5
  • Built with Spring Boot, Thymeleaf, and H2 database
